Подготовить шахматку
У вас 7×10 клеток ⇒ внутренних пересечений 6×9. Измерьте сторону клетки (квадратика) в мм — это square_size_mm. Важно: меряем ровно чёрно-белую клетку, без полей.

Снять 20–30 кадров с разными ракурсами

Запустите 01_capture_chessboard_images.py

Нажимаете SPACE при хорошем ракурсе (разные углы, близко/далеко, по краям кадра).

Сохраняется в папку data/calib/.

Калибровать
Запустите 02_calibrate_camera.py с вашими параметрами (пример команд ниже). В результате получите файл, напр. calibration_cam0.npz.

Проверить
03_undistort_preview.py покажет исход/удалённую дисторсию (клавиша U), убедитесь, что прямые линии стали прямыми.

Детекция ArUco + куб
04_detect_aruco_and_cube.py — загрузит калибровку, найдёт маркер и наложит куб (по реальной стороне маркера, мм). Скрипт устойчив к разным версиям OpenCV (новое и старое API aruco) и использует solvePnP(IPPE) как основу позы.
